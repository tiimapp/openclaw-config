# EngageLab Email REST API 详细说明

## POST /v1/mail/send

### 概述
此 API 用于通过 EngageLab 平台发送电子邮件。支持单封邮件、批量邮件、模板邮件、带附件邮件等多种发送方式。

### 请求 URL
- **新加坡数据中心**: `https://email.api.engagelab.cc/v1/mail/send`
- **土耳其数据中心**: `https://emailapi-tr.engagelab.com/v1/mail/send`

### 请求方法
`POST`

### 请求头 (Headers)
| Header        | 类型   | 必填 | 描述                                       |
| ------------- | ------ | ---- | ------------------------------------------ |
| Content-Type  | String | 是   | `application/json;charset=utf-8`           |
| Authorization | String | 是   | `Basic base64(api_user:api_key)`，用于身份验证 |
