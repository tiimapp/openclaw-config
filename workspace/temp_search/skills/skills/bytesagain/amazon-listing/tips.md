# Amazon Listing — Tips

## 使用建议

1. **标题优化 (title)**: 遵循品牌+核心词+属性的结构
2. **五点描述 (bullet)**: 每条突出一个卖点，开头大写
3. **后台关键词 (backend)**: 补充标题中未出现的同义词
4. **PPC广告 (ppc)**: 从自动广告开始，逐步优化手动广告
5. **评价管理 (review)**: 合规获评，快速处理差评
6. **排名提升 (rank)**: BSR是综合指标，销量+转化率最关键

## 最佳实践

- 标题200字符以内，手机端显示约80字符
- Bullet Points每条不超过200字符
- Backend Search Terms总共249字节
- PPC预算从小开始，根据ACOS逐步调整
- 不要刷单刷评，亚马逊检测越来越严
- A+页面能提升转化率5-10%
