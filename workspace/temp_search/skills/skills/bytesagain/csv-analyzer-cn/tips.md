# CSV Data Analyzer — tips.md

## 数据分析常用技巧

### 数据清洗 Checklist
1. ✅ 检查缺失值（空单元格、NULL、N/A）
2. ✅ 检查重复行
3. ✅ 统一日期格式（YYYY-MM-DD）
4. ✅ 数字格式统一（去除货币符号、千分位）
5. ✅ 文本修剪（去除首尾空格）
6. ✅ 分类一致性（如：男/male/M统一）
7. ✅ 异常值检测（超出3σ范围）

### 可视化选择指南
| 数据类型 | 推荐图表 | 适用场景 |
|---------|---------|---------|
| 分类对比 | 柱状图 bar | 各部门销售额对比 |
| 占比分布 | 饼图 pie | 市场份额、费用占比 |
| 趋势变化 | 折线图 line | 月度增长、价格走势 |
| 相关性 | 散点图 scatter | 价格vs销量关系 |
| 分布情况 | 直方图 histogram | 年龄分布、成绩分布 |

### 常见数据问题及解决
- **编码乱码**: 用UTF-8 BOM打开，或指定GBK编码
- **日期混乱**: 2024/1/5 vs 01-05-2024 → 统一用ISO格式
- **数字存为文本**: Excel里数字左上角有绿色三角
- **合并单元格**: 拆分后用向下填充

### 数据分析四步法
1. **看** — 先浏览数据，了解结构和规模
2. **清** — 数据清洗，处理缺失和异常
3. **算** — 统计分析，找关键指标
4. **说** — 可视化呈现，讲数据故事

### CSV格式规范
```
"姓名","年龄","城市","薪资"
"张三",28,"北京",15000
"李四",32,"上海",18000
```
- 包含逗号的字段用双引号包裹
- 第一行为表头
- 编码推荐UTF-8
- 换行符用\n（Unix）或\r\n（Windows）
