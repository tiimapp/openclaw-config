# Cron Expression Tips

## 基本格式
```
┌──────── 分钟 (0-59)
│ ┌────── 小时 (0-23)
│ │ ┌──── 日 (1-31)
│ │ │ ┌── 月 (1-12)
│ │ │ │ ┌ 星期 (0-7, 0和7都是周日)
│ │ │ │ │
* * * * *
```

## 特殊字符
- `*` 任意值
- `,` 列表 (1,3,5)
- `-` 范围 (1-5)
- `/` 步长 (*/5 = 每5个单位)

## 常见陷阱
1. **日和星期冲突**: 同时指定日和星期，在标准cron中是OR关系
2. **时区问题**: cron默认使用系统时区，注意服务器时区设置
3. **AWS与Linux差异**: AWS CloudWatch cron有6个字段（多了年份），且支持`?`
4. **GitHub Actions**: 使用UTC时区，注意时差转换

## 快速参考
| 表达式 | 含义 |
|--------|------|
| `* * * * *` | 每分钟 |
| `0 * * * *` | 每小时整点 |
| `0 0 * * *` | 每天午夜 |
| `0 0 * * 0` | 每周日午夜 |
| `0 0 1 * *` | 每月1日午夜 |
| `0 0 1 1 *` | 每年1月1日午夜 |
