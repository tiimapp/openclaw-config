# Comparison Table Tips

## 对比表最佳实践

### 表格设计原则
1. **维度统一** — 所有选项用同样的维度评估
2. **数据可比** — 用相同单位和标准
3. **重点突出** — 关键差异用标记醒目显示
4. **简洁明了** — 维度不超过10个

### 评分体系
- ✅ 支持/有
- ❌ 不支持/无
- ⭐⭐⭐⭐⭐ 五星制
- 数字评分 1-10
- 💰💰💰 价格等级

### 常见对比维度
- **产品**: 价格、性能、质量、售后、品牌
- **技术**: 性能、生态、学习曲线、社区、文档
- **定价**: 月费、年费、功能、限制、客服
- **服务**: 响应时间、覆盖范围、SLA、价格

## 输出格式
- Markdown: 适合文档、README、笔记
- HTML: 适合网页、报告、分享
