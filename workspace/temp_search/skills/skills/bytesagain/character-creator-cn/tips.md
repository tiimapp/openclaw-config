# Character Creator Tips

## 角色设计原则

1. **内在矛盾** — 好角色有内在冲突（勇敢但害怕孤独）
2. **动机明确** — 角色想要什么？为什么？愿意付出什么代价？
3. **缺陷让角色真实** — 完美的角色是无聊的
4. **环境塑造人** — 成长环境影响性格和价值观
5. **行动定义角色** — "做了什么"比"是什么人"更重要

## 性格模型

| 模型        | 适用场景            |
|------------|-------------------|
| MBTI 16型   | 快速定义性格基调     |
| 大五人格     | 更科学细腻的描述     |
| 九型人格     | 深层动机和恐惧      |
| 四气质说     | 简单直观的分类      |

## 角色弧光类型

- **正向弧光**：克服缺陷，变得更好（最常见）
- **负向弧光**：被压力击垮，走向堕落（悲剧角色）
- **平坦弧光**：角色不变，但改变了世界（英雄类）

## 人物关系设计

- 关系创造冲突，冲突推动故事
- 对立角色互相映照彼此的特质
- 群体角色要有明确的差异化
- 关系应该在故事中发生变化

## 角色卡片要素

- 基本信息（名字、年龄、外貌）
- 性格特征（3个优点 + 3个缺点）
- 背景故事（关键转折事件）
- 核心动机和最大恐惧
- 口头禅/标志性动作
- 与其他角色的关系
