# 丝氨酸蛋白酶已知功能域参考

## Pfam S1 家族保守块（用于精确匹配）

| 保守块 | 序列 | 功能 |
|--------|------|------|
| `GDSG` | 4aa | 含催化 Ser195，直接执行亲核攻击底物肽键，形成酰基-酶中间体 |
| `GDSGGP` | 6aa | His57 周围保守区，参与催化三联体质子转移，是催化机制核心 |
| `IVGG` | 4aa | 酶原激活位点，前体蛋白切割后 Ile16 形成新 N 端，插入激活口袋触发构象变化 |
| `GICAG` | 5aa | S1 底物结合口袋相关，影响底物特异性 |

## 主要功能域（结构层面）

| 功能域 | 典型特征序列 | 功能 |
|--------|-------------|------|
| 信号肽 | N 端疏水区（MXXXX...FCLLL...）| 蛋白分泌导向，通常15-30aa |
| Gla 域 | 富含 Glu（E），维生素K依赖 | 钙离子结合，锚定磷脂膜（凝血因子） |
| EGF 样域 | 6个保守 Cys，`CXXXXXC` 模式 | 蛋白间相互作用 |
| Kringle 域 | 3对二硫键，约80aa | 结合赖氨酸，底物/辅因子识别 |
| 丝氨酸蛋白酶域 | 含 His-Asp-Ser 催化三联体 | 执行蛋白质水解催化 |
| 纤连蛋白III型重复域 | `PXXX...GDSFTI...` 模块重复 | 细胞-基质相互作用（有袋类常见） |

## 二硫键网络

- 保守 Cys 残基成对形成二硫键，维持蛋白三维构象稳定性
- 丝氨酸蛋白酶家族通常含多个二硫键（数量因物种/亚家族而异）
- 大量 Cys（>50个）提示该物种丝氨酸蛋白酶包含多个含二硫键的结构域

## 功能预测的保守序列线索

| 特征 | 功能推断 |
|------|---------|
| N 端 MXXXX...FCILL/FLLL | 信号肽 + 分泌导向 |
| Tyr/Pro 富集重复（`PGYXPGY...`）| 胶原蛋白样/底物识别延伸区 |
| Arg/Lys 富集 | 底物结合正电荷区域（识别 Arg/Lys 底物） |
| 含跨膜螺旋（XXXFLLL/XXXXVLL）| 膜锚定型蛋白酶 |
| 重复模块（>100aa 重复出现）| 锚蛋白重复域 / Fibronectin III 重复域 |
