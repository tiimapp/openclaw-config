# Smart Search 决策参考

**这不是一个 skill，而是搜索引擎选择的决策参考文档。**

---

## 🎯 双引擎策略

| 引擎 | 定位 | Token/次 | 使用场景 |
|------|------|---------|----------|
| **SearXNG** | 全面结果 + 免费 + 默认 | ~350 | 日常搜索、深度调研、快速查询 |
| **Tavily** | AI 内容生成 + 摘要 + 快速 | ~950 | 写文案、小红书、公众号、报告 |

---

## 📋 决策规则

### 用 Tavily 的场景（AI 内容生成）
```
✅ 写文案、写文章、写报告、写代码
✅ 生成内容、生成文章
✅ 小红书、公众号、脚本
✅ 中等长度问题（6-15 词），需要摘要
```

### 用 SearXNG 的场景（默认引擎）
```
✅ 日常查询、简单问题（≤5 词）
✅ 深度调研、详细分析、全面了解
✅ 快速搜索、大概了解
✅ 复杂问题（>15 词），需要全面结果
```

### 用户指定优先
```
"用 searxng 搜索 XXX" → SearXNG
"用 tavily 搜索 XXX" → Tavily
```
