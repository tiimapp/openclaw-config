# HiVulse for OpenClaw

**AI-powered professional documentation generator** — 远超 Claude/Cursor 的代码文档技能

[<image-card alt="ClawHub" src="https://img.shields.io/badge/Install%20on-ClawHub-blue" ></image-card>](https://www.clawhub.ai/skills/hivulse-docs)

### 为什么程序员需要这个技能？

Claude 和 Cursor 生成的文档常常：
- 内容 hallucination、结构混乱
- 缺少 UML 图和变更日志
- 每次都要手动 copy-paste

**Hivulse** 一键解决：自动解析整个 codebase，输出企业级专业文档（带图、目录、变更记录），支持 Markdown/Word/PDF，自动同步。
