# 宏观经济 / 国内宏观

---

#### 居民消费价格指数

接口：`list_economic_cn_cpi`　`GET`

查询中国居民消费价格指数（CPI）历史数据，支持按起止日期（格式为yyyy-MM-dd）筛选和分页，返回指定时间段内每个统计截止日期的当月CPI数据、累计CPI数据、同比和环比变化率，用于分析通货膨胀趋势和宏观经济状况。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py economic/cn-cpi
```

#### 工业出厂价格分类指数

接口：`list_economic_cn_ppi`　`GET`

查询中国工业生产者出厂价格指数（PPI）的历史数据，支持按开始日期（beginDate）和结束日期（endDate）筛选，返回指定时间段内各统计截止日期（date）的当月数据（current_month_data）、累计数据（cumulative_data）、当月同比（yoy）、当月环比（mom）及累计同比（yoy_acc）等核心指标，用于宏观经济分析和价格趋势监测。日期格式为 yyyy-MM-dd。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py economic/cn-ppi
```
