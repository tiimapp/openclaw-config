## 高级查询

### 示例 4：不同时间窗口的热点

**用户输入**：
```
过去 6 小时的热点话题
```

**OpenClaw 行为**：
1. 调用聚合 API，设置 window=6h
2. 获取更长时间范围的聚合数据
3. 返回综合分析结果

### 示例 5：特定主题快讯

**用户输入**：
```
给我看看关于比特币的最新消息
```

**OpenClaw 行为**：
1. 调用快讯 API 获取数据
2. 过滤包含 "比特币" 或 "Bitcoin" 的快讯
3. 返回相关结果

### 示例 6：组合查询

**用户输入**：
```
今天有什么重要的加密货币新闻？同时告诉我热点话题和 Polymarket 预测
```

**OpenClaw 行为**：
1. 同时调用快讯、聚合和预测事件 API
2. 整合三部分数据
3. 返回综合报告

## 最佳实践

### 1. 明确的查询

✅ **好的查询**：
- "最新的比特币快讯"
- "Polymarket 上关于选举的预测"
- "今天的加密货币新闻"

❌ **不够明确的查询**：
- "新闻"（太宽泛）
- "告诉我一些东西"（不明确）

### 2. 组合使用

可以将 ME News Skill 与其他 Skills 组合使用：

```
用户：查询比特币价格，然后告诉我相关的最新快讯

OpenClaw：
1. [使用价格查询 Skill] 当前比特币价格：$70,123
2. [使用 ME News Skill] 相关快讯：比特币突破 $70,000...
```

### 3. 定期更新

建议定期查询以获取最新信息：

```
用户：每天早上 9 点给我一份加密货币资讯简报

OpenClaw：好的，我会每天早上 9 点为您准备加密货币资讯简报。
```

## 技巧与窍门

1. **使用触发词**：明确提到 "ME News"、"快讯"、"热点"、"Polymarket" 可以更快激活 Skill

2. **指定数量**：可以要求特定数量的结果
   ```
   "给我看 10 条最新快讯"
   "前 5 个热门代币"
   ```

3. **时间范围**：可以询问特定时间的新闻和热点
   ```
   "今天的快讯"
   "最近一小时的热点"
   "过去 6 小时的热门代币"
   ```

4. **主题过滤**：可以指定感兴趣的主题
   ```
   "关于 DeFi 的快讯"
   "NFT 相关的预测市场"
   "比特币的市场情绪"
   ```

5. **情绪分析**：利用聚合功能了解市场情绪
   ```
   "哪些代币看涨？"
   "市场对以太坊的情绪如何？"
   ```
