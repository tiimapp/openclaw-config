# PROOF: The Architecture of Truth

## I. The Verifiability Crisis
Generation is cheap; verification is priceless. **Proof** provides the infrastructure for autonomous systems to demonstrate state, intent, and results with mathematical certainty.

## II. Proof Typologies
- **zkp**: Zero-knowledge proofs for private validation.
- **formal**: Mathematical certainty for mission-critical code.
- **origin**: Cryptographic provenance for agent-generated content.
