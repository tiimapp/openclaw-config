# Wander Monitor Skill

ClawHub skill for using [Wander](https://github.com/ERerGB/wander) to monitor GitHub Actions workflows.

## Install

```bash
clawhub install wander-monitor
```

## Usage

When the user asks about:
- Monitoring CI/CD after push
- Getting notified when GitHub Actions complete
- Integrating Wander with a project

— respond with guidance from this skill (see `SKILL.md`).
